# Team_Smash
# 🌱 Hyperlocal Circular Economy Platform

A full-stack sustainability platform that enables local businesses to exchange reusable waste resources, track environmental impact using AI analytics, and promote circular economy practices.

## 🚀 Problem
Local businesses generate reusable waste such as coffee grounds, food scraps, and packaging materials. Due to the lack of a real-time exchange system, these resources often end up in landfills, causing environmental damage and economic loss.

## 💡 Solution
This platform connects businesses in a hyperlocal network to list, discover, and exchange waste resources while tracking sustainability impact like CO₂ savings, landfill diversion, and economic value generated.

## ✨ Key Features
- Business login and resource listing marketplace  
- Resource exchange request and completion workflow  
- AI-based sustainability impact analytics  
- Interactive dashboard with trend graphs  
- Simulated blockchain transaction logging  

## 🏗️ Tech Stack
- **Frontend:** React (Vite), Axios, Recharts  
- **Backend:** Node.js, Express.js, MongoDB Atlas  
- **AI Service:** Python (Flask)  
- **Deployment:** Vercel (Frontend), Render (Backend)

## 🔄 User Flow
Login → Post Resource → Request Exchange → Complete Exchange → View Impact Dashboard

## 🌍 Impact
- Reduces landfill waste  
- Lowers carbon footprint  
- Promotes circular economy practices  
- Creates economic value from waste  

## ⚙️ Local Setup

### Backend
```bash
cd backend
npm install
npm start
```

Create `.env` file:
```
MONGO_URI=your_mongodb_uri
AI_URL=your_ai_service_url
```

### Frontend
```bash
cd frontend
npm install
npm run dev
```
